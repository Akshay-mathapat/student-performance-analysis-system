🎓 Student Performance Analysis System

A Streamlit-based web application to manage and analyze student academic performance.
The system allows users to add student records, analyze marks, calculate grades, find toppers, and generate statistical insights using pure Python.

Built with Streamlit and pandas.

📌 Features

✅ Add student records
✅ View all student data in table format
✅ Search students by ID
✅ Calculate total, average, and grade
✅ Identify class topper
✅ Subject-wise average analysis
✅ Grade distribution visualization
✅ Standard deviation calculation
✅ Interactive dashboard UI

🧠 Technologies Used

Python

pandas

⚙️ Installation

Clone the repository:

git clone https://github.com/yourusername/student-performance-analysis-system.git

Navigate to the project folder:

cd student-performance-analysis-system

Install dependencies:

pip install -r requirements.txt
▶️ Run the Application

Start the Streamlit app:

streamlit run app.py

The application will open in your browser:

http://localhost:8501
